# PHP Basics

This repository contains my hands-on practice while learning PHP fundamentals
with a focus on WordPress development.

##  Topics Covered
- PHP variables and arrays
- Loops (for, foreach)
- Writing reusable functions

##  File Structure

- arrays.php – working with arrays
- functions.php – function examples
- loops.php – loop examples

## Purpose
To build strong PHP fundamentals before moving into WordPress plugin development.
